# Bot Creation User Experience

## Design Philosophy

### Recruitment Metaphor
Bot creation should feel like **recruiting a specialized intelligence operative** rather than configuring a generic automation tool. The UX emphasizes:
- **Mission-focused selection** over technical configuration
- **Capability assessment** over parameter tuning  
- **Operational fit** over feature lists
